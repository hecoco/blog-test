## flex

```css
display: flex; /* inline-flex; */
```

```css
改变文档流方向/改变主轴方向  
flex-direction: row | row-reverse | column | column-reverse  
从右到左 | 从左到右 | 从上到下 | 从下动上 改变文档流方向 

是否折行/换行  
flex-wrap: wrap | wrap-reverse 默认nowrap  

flex-flow 缩写 flex-direction flex-wrap
```

```css
左 | 右 | 居中 | 平均分配  

align-items: center 横向居中
```
